 Key Features
Intelligent Dashboard: High-level overview of active applications, AI screening adoption rates, and "stale" application alerts.

AI-Assisted Screening: Visualizes AI-generated scores and priority levels for candidates to streamline the shortlisting process.

Dynamic Pipeline Management: Interactive tabs to filter applications by stage (New, Interview, Offer, Closed).

Offer Approval Workflow: Dedicated module for reviewing salary bands, proposed offers, and compliance status.

Portable Design: Entire application logic, styling, and mockup data are contained within a single HTML file for instant deployment and preview.

🛠️ Tech Stack
HTML5: Semantic structure for the dashboard and modals.

CSS3: Custom properties (CSS variables), Flexbox, and Grid for a responsive, modern layout. Includes specialized components like "Slide Panels" and "Toast Notifications."

JavaScript (Vanilla): Client-side routing (page switching), dynamic table rendering, search/filter logic, and state management for mock data.

Typography: Google Fonts integration featuring Geist and Instrument Serif.
